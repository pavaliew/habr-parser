Система для анализа комментариев пользователей Habr.

## Ключевые возможности
* Веб-парсинг: Сбор комментариев пользователя со страницы его профиля на Habr.
* Генерация облака слов: Визуализация наиболее часто употребляемых слов в комментариях для быстрого обзора основных тем.
* Анализ тональности (Sentiment Analysis): Определение эмоциональной окраски каждого комментария (позитивная, негативная, нейтральная) с использованием предобученной модели rubert-tiny-sentiment-balanced из библиотеки Hugging Face Transformers.
* Статистический отчет: Расчет и вывод процентного соотношения комментариев по их тональности.
* Асинхронность: Бот построен на библиотеке aiogram и работает асинхронно, что позволяет ему эффективно обрабатывать запросы.

## Используемые технологии
* Язык: Python 3.12
* Телеграм-бот: aiogram 3.x
* Веб-парсинг: BeautifulSoup4, requests
* Машинное обучение: Hugging Face Transformers, PyTorch
* Генерация изображений: WordCloud, Matplotlib
* Конфигурация: python-dotenv
* Управление зависимостями: uv

## Установка и запуск
0. Установите uv
```bash
# Для macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Для Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/your-repo-name.git
```
2. Создайте виртуальное окружение и установите зависимости
```bash
uv venv
uv pip install -r requirements.txt
```
3. Настройте переменные окружения, создав файл .env с токеном бота
```text
BOT_TOKEN="123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
```
4. Запустите бота
```bash
uv run python main.py
```

## Структура проекта
```text
/
|-- main.py           # Главный файл: запуск и логика бота (aiogram)
|-- parser.py         # Модуль для парсинга HTML-страницы Habr
|-- analyzer.py       # Модуль для анализа тональности комментариев (ML)
|-- wordcloud_gen.py  # Модуль для генерации облака слов
|-- utils.py          # Вспомогательные функции (например, работа с файлами)
|-- test.py           # Модульные тесты (опционально, но рекомендуется)
|-- temp/             # Папка для временных файлов (например, изображений)
|-- .env              # Файл с переменными окружения (не для Git)
|-- requirements.txt  # Список зависимостей
|-- README.md         # Этот файл
```
